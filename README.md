# README Kies Op Feiten

Voor dit groepsproject ontwikkelden we een informatieve website over de huidige en vorige Tweede Kamerverkiezingen, zodat mensen gemakkelijk informatie daarover konden vinden. Het project is gebouwd met Vue.js, Java, Spring boot docker.

## Werkende functies in de Kies op feiten website

- Inlog/register pagina
- Provincie, gemeente en kieskring kaart om tweede kamer resultaten te zien per regio
- Discussie pagina om eigen discussies te starten en zien
- Coalitiemaker
- Lijst van kanidaten en info zien per partij
- Stemwijzer om je passende partij te vinden
- Account pagina
- Pollingstation
