# Style Guide Document

## 1. Logo
- **Versions**:
    - Full color (main logo)
    - Icon version (favicon and mobile use)
- **Usage**:
    - The full logo must always be used in the header on all pages.
    - The icon version can be used in the footer or as a favicon.
- **Dimensions & spacing around it**:
    - Height: `50px`
    - Minimum spacing: `20px` clear space around logo

---

## 2. Color Palette
- **Primary color(s)**:
    - HEX: #ffffff
    - Application: (e.g. headings, background)

- **Secondary color(s)**:
    - HEX: #1a3aff
    - Application: (e.g. accents, links, buttons, icons)

- **Background color**: background-color: #ffffff;
- **Text color**: color: #000000;

---

## 3. Typography
- **Main font**: red hat display
    - Usage: titles, headings (H1, H2, H3)
- **Secondary font**: red hat display
    - Usage: body text, forms
- **Font size & weight**:
    - H1: 32px, bold
    - H2: 24px, semi-bold
    - Body: 16px, regular

---

## 4. Images & Icons
- **Style**: minimalistic
- **Usage**: (stock photos, own photos, icons)
- **Filters/Editing**: use stock image, but add radius to image to make it a bit rounder 
border-radius: 8px;

---

## 5. Buttons & UI Elements
- **Primary button**:
    - Background color: #1a3aff
    - Text color: #ffffff
    - Border radius: 8px

- **Secondary button**:
    - Background color: ##DBEAFE
    - Text color: #1a3aff

---

## 6. Writing Style (Tone of voice)
- **Style**: 
- Friendly and clear
- Informal but respectful
- Short sentences, active voice

- **Examples**:
    - ✅ Do: “Klik hier om te beginnen.”
    - ❌ Don’t: “Gelieve op deze knop te drukken om het proces te starten.”

---

## 7. Layout
- **Container width**: max 1200px, centered
- **Margins & paddings**: 20px between sections
