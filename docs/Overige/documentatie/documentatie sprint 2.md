## User Stories
- **Login status tonen:** Als gebruiker wil ik rechtsboven zien of ik ben ingelogd, met naam en tijd; als ik niet ben ingelogd, kan ik inloggen.
- **Multilanguage support:** Als gebruiker wil ik de website in meerdere talen kunnen gebruiken.
- **Dark mode:** Als gebruiker wil ik de website in dark mode kunnen gebruiken (environment variable: voorkeur gebruiker).
- **Colorblind mode:** Als gebruiker wil ik de website toegankelijk voor kleurenblinden kunnen gebruiken.
- **Tijdzone aanpassing:** Als gebruiker wil ik dat tijden op de website aangepast worden naar mijn lokale tijdzone.
- **Detailniveau content & login:** Meer user stories per pagina met detail over content, login en interacties.  
> Opmerking: elke user story moet in één dag af te ronden zijn.
